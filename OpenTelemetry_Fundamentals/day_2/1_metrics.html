Metrics Concepts Using Ecommerce Application

Learn Counter, Gauge, Histogram, Exemplars, RED and USE Metrics

What Are Metrics?

Metrics are numerical measurements collected from an application over time.

Imagine an ecommerce application having:

  • Home page
  • Add to Cart API
  • Checkout API
  • Payment API

Business and engineering teams want answers like:

  • How many customers are adding products to cart?
  • How many payments are failing?
  • How slow is checkout?
  • How much memory is the server using?

All these answers come from metrics.

Application Activity → Numbers → Time Series → Dashboard

1. Counter Metric

Definition

A counter is a metric value that only increases. It starts from zero and keeps going upward.

Counters are used when we want to count how many times something happened.

Ecommerce Example: A customer clicks "Add To Cart". Every click increases the counter by one.
cart_count_total 10:00 AM = 100 carts 10:10 AM = 250 carts 10:20 AM = 600 carts Value only increases

Good Counter Examples

  • Total HTTP requests
  • Total checkout attempts
  • Total successful payments
  • Total failed payments
  • Total user registrations

Wrong Counter Usage

Current logged-in users should NOT be a counter because users can login and logout. That value goes up and down. Use Gauge instead.

Real Production Question

Question: How many orders happened in last 5 minutes? Metric: increase(order_count_total[5m])

2. Gauge Metric

Definition

A gauge represents a value that can increase or decrease. Think about a thermometer. Temperature can go up and down.

Ecommerce Example: Number of active users currently shopping. Morning: 100 users Afternoon: 500 users Night: 50 users
active_users 10 AM = 100 1 PM = 500 11 PM = 50 Moves both directions

Good Gauge Examples

  • Current active users
  • CPU percentage
  • Memory usage
  • Database connections
  • Queue size
  • Items currently in cart

Counter vs Gauge

Counter Gauge
Total orders today Current active users
Total errors happened Current CPU usage
Always increases Increase and decrease

3. Histogram Metric

Definition

Histogram measures how values are distributed. Mostly used to measure latency.

Average is dangerous. Two systems can have same average but very different user experience.

Checkout API response times: Customer A = 100 ms Customer B = 300 ms Customer C = 5 seconds We need to understand distribution.

Histogram Buckets

checkout_duration_bucket less than 100ms = 200 users less than 500ms = 900 users less than 1 sec = 980 users less than 5 sec = 1000 users

Percentiles

P95 latency means: 95 percent users are faster than this value.

P95 checkout latency = 800ms Meaning: 95% customers checkout below 800ms 5% customers are slower

Good Histogram Examples

  • API response time
  • Database query duration
  • Payment gateway latency
  • File upload time

4. Exemplars

Problem

Metrics tell us something is wrong. But they do not explain why.

Metric says: Checkout latency increased to 5 seconds. Question: Which exact request caused it?

Exemplar Solution

An exemplar connects a metric point with a trace.

Slow Metric Point → Trace ID → Exact Request Details
checkout_latency = 5 seconds attached: trace_id="abc123" Open trace: Database call took 4 seconds

5. RED Metrics

RED is mainly used for application and microservice monitoring.

Letter Meaning Ecommerce Example
R - Rate How many requests? 1000 checkout requests/min
E - Errors How many failed? 20 payment failures
D - Duration How slow? Checkout taking 2 seconds

Checkout API Health

Rate: 500 requests/min Errors: 5 failures/min Duration: P95 latency 400ms

6. USE Metrics

USE is mainly used for infrastructure monitoring. Servers, databases and machines.

Letter Meaning Example
U - Utilization How busy? CPU running 85%
S - Saturation Too much work waiting? Request queue full
E - Errors System problems Disk failures

Final Memory Rule

Counter: How many times did it happen? Gauge: What is the current value? Histogram: How long did it take? RED: Is my application healthy? USE: Is my infrastructure healthy?
