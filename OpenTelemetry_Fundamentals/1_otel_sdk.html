Flask + OpenTelemetry SDK & API

Understanding how OpenTelemetry API and SDK work in a real Flask microservice application.

Goal of This Example

In this example, students will understand:

  • What OpenTelemetry API does
  • What OpenTelemetry SDK does
  • How telemetry data is generated
  • How spans are created
  • How traces move to observability backends
  • How Flask applications are instrumented

Step 1 - Install Required Packages

pip install flask pip install opentelemetry-api pip install opentelemetry-sdk pip install opentelemetry-exporter-otlp pip install opentelemetry-instrumentation-flask
Important: Students should understand that:
  • API package defines telemetry interfaces
  • SDK package implements telemetry processing
  • Instrumentation package automatically traces Flask

Step 2 - Basic Flask Application

from flask import Flask app = Flask(__name__) @app.route("/") def home(): return "Flask Application Running" if __name__ == "__main__": app.run(port=5000)

This is a normal Flask application without observability.

Step 3 - Add OpenTelemetry API & SDK

from flask import Flask from opentelemetry import trace from opentelemetry.sdk.trace import TracerProvider from opentelemetry.sdk.trace.export import BatchSpanProcessor from opentelemetry.exporter.otlp.proto.http.trace_exporter import OTLPSpanExporter app = Flask(__name__) # Create Tracer Provider (SDK) trace.set_tracer_provider(TracerProvider()) # Create Exporter otlp_exporter = OTLPSpanExporter( endpoint="http://localhost:4318/v1/traces" ) # Add Span Processor span_processor = BatchSpanProcessor(otlp_exporter) # Connect Processor to SDK trace.get_tracer_provider().add_span_processor(span_processor) # Create Tracer (API) tracer = trace.get_tracer(__name__) @app.route("/") def home(): # Create Span with tracer.start_as_current_span("home-request"): return "OTel Flask Application" if __name__ == "__main__": app.run(port=5000)

Understanding the Code

Component Purpose
TracerProvider Main SDK component managing traces
OTLPSpanExporter Sends telemetry data to Collector
BatchSpanProcessor Batches spans before exporting
trace.get_tracer() Creates tracer using OTel API
start_as_current_span() Creates distributed trace span

What is a Span?

A span represents a single operation inside an application.

Example: Customer opens checkout page ↓ Span Created = "checkout-request" ↓ Database query executed ↓ Payment API called ↓ Response returned

Trace

Complete request journey.

Span

Individual operation inside a trace.

Attributes

Metadata attached to spans.

Adding Custom Attributes

@app.route("/checkout") def checkout(): with tracer.start_as_current_span("checkout-api") as span: span.set_attribute("user.id", 1001) span.set_attribute("payment.method", "UPI") span.set_attribute("order.amount", 5000) return "Checkout Success"

Attributes provide detailed observability context.

Auto Instrumentation

OpenTelemetry can automatically instrument Flask applications.

from opentelemetry.instrumentation.flask import FlaskInstrumentor FlaskInstrumentor().instrument_app(app)
What happens automatically?
  • Incoming HTTP requests traced
  • Response timings captured
  • Request metadata collected
  • Error spans generated automatically
