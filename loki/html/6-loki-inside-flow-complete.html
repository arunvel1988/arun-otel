1. Distributor - First Entry Point

The first Loki component receiving logs.

Alloy sends: POST /loki/api/v1/push
Distributor checks:
  • Tenant ID
  • Rate limits
  • Labels validation
  • Which ingester should receive logs
Example:

Failure Testing

Distributor Failure

Logs cannot enter Loki. Alloy retries. No query impact.

Ingester Failure

Dangerous component. Active chunks live here. Use replication_factor=3

MinIO Failure

Ingestion may continue temporarily. Chunk upload fails. Old queries fail.

Querier Failure

Logs continue coming. Only searching affected.

Compactor Failure

Logs OK. Queries OK. Retention cleanup stops.

Remember Loki Components

Component Job
Distributor Validate and route logs
Ingester Create chunks and upload storage
Querier Read chunks and answer queries
Query Frontend Split/cache queries
Compactor Optimize and delete old data
Ruler Run LogQL alerts
