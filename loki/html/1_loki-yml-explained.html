Failure Questions Students Ask

Distributor dies?

No data loss. Alloy retries and another distributor receives.

Ingester dies?

Depends on replication factor. Production uses RF=3.

S3 dies?

Writes buffer temporarily. Queries for old data fail.

Querier dies?

Only searches affected. Logging continues.

Compactor dies?

Logs continue. Optimization stops.
