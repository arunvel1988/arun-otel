Grafana Loki Configuration Explained

First Understand Loki Job

Application creates logs
|
Alloy sends logs
|
Loki receives logs
|
Ingester keeps temporary chunks
|
Compress
|
Store in MinIO(S3)

auth_enabled

auth_enabled: false

Controls Loki multi-tenancy.

false means:

No tenant authentication. Everything goes into default tenant: fake

In production:

Tenant A | logs Tenant B | logs

server

server: http_listen_port: 3100

Starts Loki HTTP server.

Example:
Grafana | | http://loki:3100 | | Loki

Port 3100 exposes:

  • Push API
  • Query API
  • Health check

common.path_prefix

common: path_prefix: /loki

Local working directory for Loki.

Example:
/loki | + cache | + temporary files

Remember: Final logs are NOT stored here. They go to MinIO.

Replication Factor

replication_factor: 1

How many Loki ingesters keep a copy.

Development:
Log | v Ingester-1 (one copy)
Production:
Log | +---- Ingester-1 | +---- Ingester-2 | +---- Ingester-3

Ring Configuration

ring: kvstore: store: inmemory

Ring tells Loki: "Which ingester owns which logs?"

Distributor receives log
asks Ring
Which ingester?
Send there

Production uses:

Consul etcd memberlist

schema_config

schema: v13 store: tsdb object_store: s3

Defines HOW Loki stores logs.

Labels | v TSDB Index

Actual Logs | v S3 / MinIO chunks

Index Configuration

index: prefix: index_ period: 24h

Index is Loki's map.

Example:
service="payment" | | chunk-123

Without index Loki cannot find logs quickly.

storage_config AWS

endpoint: minio:9000 bucketnames: loki-data

This tells Loki: Where should I store chunks?

Loki | | MinIO bucket: loki-data

Ingester

chunk_idle_period: 30s max_chunk_age: 1m

Ingester is Loki memory buffer.

New Log | RAM Chunk | wait | Compress | Upload MinIO

chunk_idle_period

chunk_idle_period: 30s

Meaning: If no new logs arrive for 30 seconds, close this chunk.

Example:
10:00:00 log 10:00:05 log (no activity) 10:00:35 Flush chunk

max_chunk_age

max_chunk_age: 1m

Maximum time a chunk can stay open.

Example:
Logs continuously coming Still after 1 minute force close chunk send MinIO

chunk_retain_period

chunk_retain_period: 5s

After uploading to MinIO, keep copy in RAM briefly.

Why?
Recent query: RAM faster than S3

limits_config

reject_old_samples:false

Accept older timestamp logs. Useful for demos.

Example:
Yesterday log arrives today Accept it

Compactor

compaction_interval:1m

Compactor cleans and optimizes index.

Many small indexes | v Compactor | v Optimized index

Querier

max_concurrent:4

How many queries Loki runs together.

Example:
4 users searching logs OK 5th waits
