What Is The Ring?

Ring = Loki cluster membership table

It stores:

  • Which ingesters are alive
  • Which ingester owns which token range
  • Where logs should be routed
Example:
RING TABLE Ingester-1 Token Range: 0 - 3000 Ingester-2 Token Range: 3001 - 7000 Ingester-3 Token Range: 7001 - 9999

Who Uses Ring?

Component Use
Ingester Registers itself and heartbeat
Distributor Finds where to write logs
Querier Finds ingesters containing recent logs

Final Memory Trick

Ring is Loki's phone directory. Ingester says: "I am available" Distributor asks: "Where should this log go?" Querier asks: "Who may have fresh logs?" Ring stores ownership, not log data.
